The basic declaration and expression exercise program list has been given bellow. All the programs has been solved in three programming languages are C, C++ and Java. Start your journey with C,C++ and Java programming with the basic and expression program list-
## Basic Declaration & Expression Exercise Program List
1. Write a C program to print Hello World.
2. Write a C program to perform input/output of all basic data types.
3. Write a C program that will print your name, your father and mother name in three separate lines
4 Write a C program to print your name, date of birth and mobile number.
5. Write a C program to convert a string to an integer, unsigned long integer, long integer and double.
6. Write a C program to return the absolute value of an integer.
7. Write a C program to find out the square of a given number.
8. Write a C Program to input any Capital letter and display it with small letter
9. Write a C program that will take three numbers from user and find their average.
10. Write a C program to enter two numbers and perform all arithmetic operations.
11. Write a C program to find the quotient and remainder of a division
12. Write a C program to declare two integers and one float variables then initialize them to 10, 15, and 12.6. It then prints these values on the screen.
13. Write a C program to perform addition, subtraction, multiplication and division of two numbers.
14. Write a C Program to accept any character from user and display its ASCII number on screen.
15. Write a C program to compute the perimeter and area of a rectangle with a height of 7 inches. and width of 5 inches.
16. Write a C program that calculates the volume of a sphere.
17. Ramesh’s basic salary is input through the keyboard. His dearness allowance is 40% of basic salary, and house rent allowance is 20% of basic salary. Write a program to calculate his gross salary.
18. The distance between two cities (in km.) is input through the keyboard. Write a program to convert and print this distance in meters, feet, inches and centimeters.
19. Write a C program that takes hours and minutes as input, and calculates the total number of minutes.
20. Write a program in C that takes minutes as input, and display the total number of hours and minutes.
21. If the marks obtained by a student in five different subjects are input through the keyboard, find out the aggregate marks and percentage marks obtained by the student. Assume that the maximum marks that can be obtained by a student in each subject is 100.
22. Write a program to convert this temperature into Centigrade degrees.
23. Write a C program to find the third angle of a triangle if two angles are given.
24. Write a program to calculate the area & perimeter of the rectangle, and the area & circumference of the circle.
25. Write a C Program to interchange values of two numbers using third variable.
26. Write a C Program to interchange values of two numbers without using third variable
27. If a five-digit number is input through the keyboard, write a program to calculate the sum of its digits. (Hint: Use the modulus operator ‘%’) (without loop)
28. If a five-digit number is input through the keyboard, write a program to reverse the number. (without loop)
29. If a four-digit number is input through the keyboard, write a program to obtain the sum of the first and last digit of this number. (without loop)
30. In a town, the percentage of men is 52. The percentage of total literacy is 48. If total percentage of literate men is 35 of the total population, write a program to find the total number of illiterate men and women if the population of the town is 80,000.
