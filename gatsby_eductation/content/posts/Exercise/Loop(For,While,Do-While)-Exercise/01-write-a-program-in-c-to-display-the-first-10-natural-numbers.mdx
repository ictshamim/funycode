## C Implementation
### Program-1: Write a program in C to display the first 10 natural numbers.
```c
//Write a C to display the first 10 natural numbers.
#include <stdio.h>
/*Write a program in C to display the first 10 natural numbers.*/
/*Using for loop*/
#include<stdio.h>
void main()
{
   int i;
   for(i=1;i<=10;i++)
   {
      printf("%d ",i);
   }
}
/*Using while loop*/
#include<stdio.h>
void main()
{
   int i=1;
   while(i<=10)
   {
      printf("%d ",i);
      i++;
   }
}
/*Using while loop*/
#include<stdio.h>
void main()
{
   int i=1;
   do
   {
      printf("%d ",i);
      i++;
   }
   while(i<=10);
}
```
### Output
```
Hello, World!
```
