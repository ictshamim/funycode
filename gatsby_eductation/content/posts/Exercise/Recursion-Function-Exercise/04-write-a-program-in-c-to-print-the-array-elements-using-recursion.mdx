## C Implementation
### 4. Write a C to print the array elements using recursion.
```c
//Write a C to print the array elements using recursion.
#include<stdio.h>
int main(){
    int k[]={1,2,3,4,5};
    arrayPrint(sizeof(k)/sizeof(int),k,0);
}
int arrayPrint(int length,int k[],int i){
    if(length<=i){
        return 0;
    }
    printf("%d",k[i]);
    arrayPrint(length,k,++i);
}
```
### Output
```
12345
```
