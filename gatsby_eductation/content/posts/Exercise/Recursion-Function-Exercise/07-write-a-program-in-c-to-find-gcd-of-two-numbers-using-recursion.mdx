## C Implementation
### 7. Write a C program to count the digits of a given number using recursion.
```c
//Write a C program to count the digits of a given number using recursion.
#include<stdio.h>
int main(){
    int one,two;
    printf("GCD of Two numbers\nEnter first number : ");
    scanf("%d", &one);
    printf("Enter second number : ");
    scanf("%d", &two);
    int ans=0;
    if(one>two)
    {
       ans= gcd(one,two);
    }
    else{
       ans= gcd(two,one);
    }
    printf("%d",ans);
}
int gcd(int one,int two){
    if(one%two==0)
    {
        return two;
    }
    else{
        gcd(two,one%two);
    }
}
```
### Output
```
GCD of Two numbers
Enter first number : 3
Enter second number : 4
1
```
