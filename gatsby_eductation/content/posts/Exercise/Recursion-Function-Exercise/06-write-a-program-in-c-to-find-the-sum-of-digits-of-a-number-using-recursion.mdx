## C Implementation
### 6. Write a C program to find the sum of digits of a number using recursion.
```c
//Write a C program to find the sum of digits of a number using recursion.
#include<stdio.h>
int main(){
    int n;
    printf("Enter a Number : ");
    scanf("%d", &n);
    printf("Sum of digits : ");
    int k=sumOfDigit(n);
    printf("%d",k);
}
int sumOfDigit(int n){
    if(n==0)
    {
        return 0;
    }
    return  (n%10)+sumOfDigit(n/10);
}
```
### Output
```
Enter a Number : 5
Number of digits : 1
```
