## C Implementation
### 8. Write a C program to get the largest element of an array using recursion.
```c
//Write a C program to get the largest element of an array using recursion.

#include<stdio.h>
int main(){
    int k[]={1,2,30,4,5};
    printf("%d",arrayPrint(sizeof(k)/sizeof(int),k,0,0));
}
int arrayPrint(int length,int k[],int i,int large){
    if(length<=i){
        return large;
    }
    if(large<k[i])
    {
        large=k[i];
    }
    arrayPrint(length,k,++i,large);
}
```
### Output
```
30
```
