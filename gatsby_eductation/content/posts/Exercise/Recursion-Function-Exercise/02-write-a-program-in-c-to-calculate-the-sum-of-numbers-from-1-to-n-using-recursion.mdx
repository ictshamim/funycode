## C Implementation
### 2. Write a C program to find the square of any number using the function.
```c
//Write a C program to find the square of any number using the function.
#include<stdio.h>
int main(){
    int n;
    printf("Enter n Number : ");
    scanf("%d", &n);
    int a= sum(n);
    printf("Sum of n number : %d",a);
}
int sum(int n){
    if(n!=0)
    {
        return n+sum(n-1);
    }
    else{
        return n;
    }
}
```
### Output
```
Enter n Number : 10
Sum of n number : 55
```
