## C Implementation
### Program-13: Write a C program to enter three side of triangle and check whether the triangle valid or not.
```c
/*Write a C program to enter three side of triangle and check whether the triangle valid or not.*/
#include<stdio.h>
void main()
{
    int a,b,c;
    scanf("%d %d %d",&a,&b,&c);
    if((a+b>c&&b+c>a&&c+a>b)&&(a>0&&b>0&&c>0))
        printf("triangle is valid");
    else
        printf("triangle is invalid");
}
```
### Output
```
Enter three angles of a triangle-
5
6
8
Triangle is valid
```
