## C Implementation
### Program-9: Write a C program to input a character and check that it’s a vowel or a consonant.
```c
/*Write a C program to input a character and check that it’s a vowel or a consonant.*/
#include<stdio.h>
void main()
{
    char s,a,e,i,o,u,A,E,I,O,U,x,y;
    scanf("%c",&s);
    x=(s=='a'||s=='e'||s=='i'||s=='o'||s=='u');
    y=(s=='A'||s=='E'||s=='I'||s=='O'||s=='U');
    if(x||y)
        printf("the  letter is vowel");
    else
        printf("the letter is consonant");
}
```
### Output
```
a
the letter is vowel
```
