## C Implementation
### Program-23: Write a C program to calculate the root of a Quadratic Equation.
```c
/*Write a C program to calculate the root of a Quadratic Equation.*/
#include<stdio.h>
void main()
{
    int r,a,b,c,x1,x2;
    printf("Enter value of a,b,c-");
    scanf("%d%d%d",&a,&b,&c);
    r=b*b-4*a*c;
    if(r>0)
    {
        x1=(-b+sqrt(r))/2*a;
        x2=(-b-sqrt(r))/2*a;
        printf("x1=%d x2=%d",x1,x2);
    }
    if(r==0)
    {
        x1=-b/2*a;
        x2=-b/2*a;
        printf("x1=%d x2=%d",x1,x2);
    }
    if(r<0)
        printf("math is error");
}
```
### Output
```
Enter value of a,b,c-
1
4
2
x1=0
x2=-3
```
