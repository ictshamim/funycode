## C Implementation
### Program-21: Write a C program to check whether a number is negative, positive or zero.
```c
/*Write a C program to check whether a number is negative, positive or zero.*/
#include<stdio.h>
void main()
{
    int a;
    printf("Enter a Number:");
    scanf("%d",&a);
    if(a>0)
        printf("number is positive number ");
    else if(a<0)
        printf("number is negative number");
    else
        printf("number is equal to zero");
}
```
### Output
```
Enter a Number:
50
Positive number
```
