## C Implementation
### Program-14: Write a C program to enter three angle of triangle and check whether the right triangle or not.
```c
/*Write a C program to enter three angle of triangle and check whether the right triangle or not.*/
#include<stdio.h>
void main()
{
    int a,b,c;
    printf("enter three angles of a triangle-");
    scanf("%d %d %d",&a,&b,&c);
    if((a==90&&(b>0&&c>0&&b+c==90))||(b==90&&(a>0&&c>0&&a+c==90))||(c==90&&(a>0&&b>0&&a+b==90)))
        printf("right triangle");
    else
        printf("not right triangle");
}
```
### Output
```
enter three angles of a triangle-
90
45
45
triangle is Right
```
