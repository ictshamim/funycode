## C Implementation
### Program-5: Write a C program to input a number and check whether it is positive or negative.
```c
/*Write a C program to input a number and check whether it is positive or negative.*/
#include<stdio.h>
int main()
{
     int a;
    Printf("Enter a Number:");
    scanf("%d",&a);
    if(a>0)
        printf("positive number");
    if(a<0)
        printf("negative number");
    if(a==0)
        ("the number is equal to zero");
}
```
### Output
```
Enter a Number:
25
positive number
```
