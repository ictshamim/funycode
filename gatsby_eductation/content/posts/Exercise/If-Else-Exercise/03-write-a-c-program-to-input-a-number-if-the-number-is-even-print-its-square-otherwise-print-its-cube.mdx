## C Implementation
### Program-3: Write a C program to input a number. If the number is even, print its square otherwise print its cube.
```c
/*Write a C program to input a number. If the number is even, print its square otherwise print its cube.*/
#include<stdio.h>
int main()
{
    int a;
    printf("Enter a number");
    scanf("%d",&a);
    if(a%2==0)
        printf("square of the number=%d",(a*a));
    else
        printf("Cube of the number=%d",(a*a*a));
}
```
### Output
```
Enter a number
10
square of the number=100
```
