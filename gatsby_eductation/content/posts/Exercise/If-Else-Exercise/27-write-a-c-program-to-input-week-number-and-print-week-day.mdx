## C Implementation
### Program-27: Write a C program to input week number and print week day.
```c
/*Write a C program to input week number and print week day.*/
#include<stdio.h>
void main()
{
    int n;
    printf("Enter a Day:");
    scanf("%d",&n);
    if(n==1)
        printf("Monday");
    else if(n==2)
        printf("Tuesday");
    else if(n==3)
        printf("Wednesday");
    else if(n==4)
        printf("Thursday");
    else if(n==5)
        printf("Friday");
    else if(n==6)
        printf("Saturday");
    else if(n==7)
        printf("Sunday");
    else
        printf("not");
}
```
### Output
```
Enter a Day:
4
Thursday
```
