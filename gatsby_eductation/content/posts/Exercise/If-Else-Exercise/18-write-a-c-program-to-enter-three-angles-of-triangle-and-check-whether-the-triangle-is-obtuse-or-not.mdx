## C Implementation
### Program-18: Write a C program to enter three angles of triangle and check whether the triangle is Obtuse or not.
```c
/*Write a C program to enter three angles of triangle and check whether the triangle is Obtuse or not.*/
#include<stdio.h>
void main()
{
    int a,b,c;
    printf("Enter three angles of a triangle-");
    scanf("%d %d %d",&a,&b,&c);
    if((a>90||b>90||c>90)&&(a+b+c==180))
        printf("triangle is obtuse triangle");
    else
        printf("triangle is not obtuse triangle");
}
```
### Output
```
Enter three angles of a triangle-
120
30
30
triangle is obtuse triangle
```
