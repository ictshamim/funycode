## C Implementation
### Program-16: Write a C program to enter three angles of triangle and check whether the triangle is Acute or not.
```c
/*Write a C program to enter three angles of triangle and check whether the triangle is Acute or not.*/
#include<stdio.h>
void main()
{
    int a,s,d;
    printf("Enter three angles of a triangle-");
    scanf("%d %d %d",&a,&s,&d);
    if((a<90&&s<90&&d<90)&&(a+s+d==180))
        printf("triangle is acute");
    else
        printf("triangle is not acute");
}
```
### Output
```
Enter three angles of a triangle-
80
80
20
triangle is acute
```
