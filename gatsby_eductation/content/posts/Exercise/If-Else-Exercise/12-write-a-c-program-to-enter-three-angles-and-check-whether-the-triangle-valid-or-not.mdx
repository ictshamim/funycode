## C Implementation
### Program-12: Write a C program to enter three angles and check whether the triangle valid or not.
```c
/*Write a C program to enter three angles and check whether the triangle valid or not.*/
#include<stdio.h>
void main()
{
    int a,b,c;
    printf("enter three angles of a triangle-");
    scanf("%d%d%d",&a,&b,&c);
    if((a+b+c==180)&&(a>0&&b>0&&c>0))
        printf(" triangle is valid");
    else
        printf("triangle is invalid");
}
```
### Output
```
enter three angles of a triangle-
20
60
80
triangle is invalid
```
