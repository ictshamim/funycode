## C Implementation
### Program-11: Write a C program to calculate area of an equilateral triangle.
```c
/*Write a C program to calculate area of an equilateral triangle.*/
#include<stdio.h>
void main()
{
    float a,x;
    printf("the value of the side of the equilateral triangle-");
    scanf("%f",&a);
    x=((sqrt(3))/4)*a*a;
    printf("area of a equilateral triangle=%f",x);
}
```
### Output
```
the value of the side of the equilateral triangle-
5
area of a equilateral triangle=10.8253
```
