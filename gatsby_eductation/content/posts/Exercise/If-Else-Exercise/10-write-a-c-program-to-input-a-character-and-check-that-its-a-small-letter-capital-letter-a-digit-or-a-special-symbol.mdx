## C Implementation
### Program-10: Write a C program to input a character and check that it’s a small letter, capital letter, a digit or a special symbol.
```c
/*Write a C program to input a character and check that it’s a small letter, capital letter, a digit or a special symbol.*/
#include<stdio.h>
void main()
{
    char a,z,A,Z;
    scanf("%c",&a);
    if(a>='A'&&a<='Z')
        printf("it is capital letter");
    else if(a>='a'&&a<='z')
        printf("it is small letter");
    else if(a>='0'&&a<='9')
        printf("it is a digit");
    else
        printf("it is a special symbol");
}
```
### Output
```
a
it is small letter
```
