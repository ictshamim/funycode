## C Implementation
### Program-2: Write a C program to input a number and check whether it is even or odd.
```c
/*Write a C program to input a number and check whether it is even or odd.*/
#include<stdio.h>
int main()
{
    int a;
    printf("Enter a number");
    scanf("%d",&a);
    if(a%2==0)
        printf("even number");
    else
        printf("odd number");
}
```
### Output
```
Enter a number
20
Even Number
```
