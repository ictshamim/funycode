## C Implementation
### Program-15: Write a C program to enter three side of triangle and check whether the triangle is Isosceles or not.
```c
/*Write a C program to enter three side of triangle and check whether the triangle is Isosceles or not.*/
#include<stdio.h>
void main()
{
    int a,b,c;
    printf("Enter three side of a triangle-");
    scanf("%d %d %d",&a,&b,&c);
    if((a==b||b==c||c==a)&& (a>0&&b>0&&c>0))
        printf("triangle is isosceles");
    else
        printf("triangle is not isosceles");
}
```
### Output
```
Enter three angles of a triangle-
45
45
45
triangle is isosceles
```
