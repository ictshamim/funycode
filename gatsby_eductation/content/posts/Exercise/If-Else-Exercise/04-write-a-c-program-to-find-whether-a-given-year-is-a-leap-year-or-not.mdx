## C Implementation
### Program-4: Write a C program to find whether a given year is a leap year or not.
```c
/*Write a C program to find whether a given year is a leap year or not.*/
#include<stdio.h>
void main(){
int n;
printf("Enter a Year: ");
scanf("%d",&n);
if(n%4==0)
    if(n%100==0)
        if(n%400==0)
            printf("leap year");
            else
                printf("not leap year");
        else
        printf("leap year");
    else
    printf("not leap year");
}
```
### Output
```
Enter a Year
2000
Leap Year
```
