## C Implementation
### Program-8:Write a C program to accept a coordinate point in a XY coordinate system and determine in which quadrant the coordinate point lies.
```c
/*Write a C program to accept a coordinate point in a XY coordinate system and determine in which quadrant the coordinate point lies.*/
#include<stdio.h>
int main()
{
    int x,y;
    printf("Enter two Point");
    scanf("%d %d",&x,&y);
    if(x>0 && y>0)
        printf("(%d,%d) in the first quadrant",x,y);
    else if(x<0 && y>0)
        printf("(%d,%d) in the second quadrant",x,y);
    else if(x<0 && y<0)
        printf("(%d,%d) in the third quadrant",x,y);
    else if(x>0 && y<0)
        printf("(%d,%d) in the fourth quadrant",x,y);
    else
        printf("(%d,%d) lies at the origin",x,y);
}
```
### Output
```
Enter two point:
4
5
4,5 in the first quadrant
```
