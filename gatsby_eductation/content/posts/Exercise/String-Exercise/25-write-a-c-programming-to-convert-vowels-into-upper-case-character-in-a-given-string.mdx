## C Implementation
### 25. Write a C program to convert vowels into upper case character in a given string.
```c
//Write a C program to convert vowels into upper case character in a given string.
#include <stdio.h>
int main()
{
   char str[255];
   int  i;
   printf("enter a string: ");
   gets(str);
   i=0;
   while(str[i]!='\0')
   {
        if(str[i]=='a' ||str[i]=='e' ||str[i]=='i' ||str[i]=='o' ||str[i]=='u')
            str[i]=str[i]-32;
        i++;
   }
   printf("converted string is:%s",str);
}
```
### Output
```
enter a character: a
This is an alphabet
```
