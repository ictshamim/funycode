## C Implementation
### 1. Write a C program to input a string and print it.
```c
//Write a C program to input a string and print it.
#include <stdio.h>
#include<string.h>
int main()
{
    char word[50];
    scanf("%s", word);
    printf("The String is %s.", word);
    return 0;
}
```
### Output
```
Hello World
The String is Hello World
```
