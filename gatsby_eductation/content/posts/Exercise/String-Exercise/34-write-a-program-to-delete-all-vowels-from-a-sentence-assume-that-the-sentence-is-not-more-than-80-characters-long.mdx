## C Implementation
### 34. Write a C program to delete all vowels from a sentence. Assume that the sentence is not more than 80 characters long.
```c
//Write a C program to delete all vowels from a sentence. Assume that the sentence is not more than 80 characters long.
#include<stdio.h>
#include<string.h>
int main()
{
    char str[100];
    char vowel[15] = {'a','e','i','o','u','A','E','I','O','U'};
    int p=0,i,j;
    printf("Enter the string: ");
    gets(str);
    int len = strlen(str);
    for(i=0;i<len;i++)
    {
       str[i]=str[i+p];
       for(j=0;j<9;j++)
       {
           if(str[i]==vowel[j]){
             p++;
             len--;
             i--;
          break;
           }
       }
    }
    str[len]='\0';
    printf("the string will be '%s' \n",str);
}
```
### Output
```
Enter the string: hello there
the string will be 'hll thr'
```
