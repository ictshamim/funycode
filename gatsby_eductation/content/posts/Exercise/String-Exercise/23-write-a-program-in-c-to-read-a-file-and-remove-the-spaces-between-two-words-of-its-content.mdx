## C Implementation
### 23. Write a C program to read a file and remove the spaces between two words of its content.
```c
//Write a C program to read a file and remove the spaces between two words of its content.
#include <string.h>

int main()
{
    char str[100];
    int  i,j=0;
    printf("Enter  the string : ");
    gets(str);
    for(i=0;str[i];i++)
    {
         str[i]=str[i+j];


         if(str[i]==' '|| str[i]=='\t')
         {
          j++;
          i--;
        }

    }
    printf("string removing spaces:");

    printf("%s",str);
}
```
### Output
```
Enter  the string : You can support us by sharing our website with your friend
string removing spaces:Youcansupportusbysharingourwebsitewithyourfriend
```
