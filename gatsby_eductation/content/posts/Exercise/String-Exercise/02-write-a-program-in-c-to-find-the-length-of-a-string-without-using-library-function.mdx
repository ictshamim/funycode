## C Implementation
### 2. Write a C program to find the length of a string without using library function.
```c
//Write a C program to find the length of a string without using library function.
#include <stdio.h>
#include <string.h>
int main()
{
    char word[50];
    int i, a = 0;
    printf("Enter a string : ");
    scanf("%s", word);
    for (i = 0; word[i] != '\0'; i++)
    {
        a++;
    }
      printf("The length is %s : %d\n\n", word, a);
}
```
### Output
```
Enter a string : hello
The length is hello : 5
```
