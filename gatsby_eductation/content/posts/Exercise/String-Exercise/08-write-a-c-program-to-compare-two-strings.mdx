## C Implementation
### 8. Write a C program to input a string and print it.
```c
//Write a C program to input a string and print it.
#include <stdio.h>
#include<string.h>
int main()
{
   char str1[20];
   char str2[20];
   int i;
   printf("Enter first string : ");
   scanf("%s",str1);
   printf("Enter second string : ");
   scanf("%s",str2);
   i=strcmp(str1,str2);
   if(i==0)
      printf("strings are equal");
   else
      printf("strings are not equal");
   return 0;
}
```
### Output
```
Enter first string : hi
Enter second string : hello
strings are not equal
```
