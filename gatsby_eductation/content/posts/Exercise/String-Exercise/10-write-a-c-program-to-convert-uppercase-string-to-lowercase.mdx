## C Implementation
### 10. Write a C program to convert uppercase string to lowercase.
```c
//Write a C program to convert uppercase string to lowercase.
#include<stdio.h>
#include<string.h>
int main(){
   char str[25];
   int i;
   printf("Enter the string: ");
   gets(str);

   for(i=0;i<=strlen(str);i++){
      if(str[i]>='A'&&str[i]<='Z')
         str[i]=str[i]+32;
   }
   printf("\nLower Case String is: %s",str);
   return 0;
}
```
### Output
```
Enter the string: HELLO
Lower Case String is: hello
```
