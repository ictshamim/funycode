## C Implementation
### 4. Write a C program to print individual characters of string in reverse order.
```c
//Write a C program to print individual characters of string in reverse order.
#include <stdio.h>
int main()
{
  char str[50];
  int i,j=0;
  printf("enter string ");
  gets(str);
  for(i=0 ;str[i] !='\0'; i++)
  j++;
  for(i=j-1;i>=0;i--)
    printf("%c ",str[i]);
}
```
### Output
```
enter string hello
o l l e h
```
