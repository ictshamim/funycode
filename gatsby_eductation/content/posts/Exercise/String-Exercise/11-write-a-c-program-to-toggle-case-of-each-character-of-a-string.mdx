## C Implementation
### 11. Write a C program to toggle case of each character of a string.
```c
//Write a C program to toggle case of each character of a string.
#include<stdio.h>
int main()
{
    char str[100];
    int i;

    printf("Enter a string: ");
    gets(str);
    for(i=0;str[i]!='\0';i++)
    {
        if(str[i]>='A' && str[i]<='Z')
            str[i]=str[i]+32;
        else if(str[i]>='a' && str[i]<='z')
            str[i]=str[i]-32;
    }

    printf("after toogle characters are: %s",str);
    return 0;
}
```
### Output
```
Enter a string: HellO
after toogle characters are: hELLo
```
