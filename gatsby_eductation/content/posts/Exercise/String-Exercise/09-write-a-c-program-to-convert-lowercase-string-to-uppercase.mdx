## C Implementation
### 9. Write a C program to convert lowercase string to uppercase.
```c
//Write a C program to convert lowercase string to uppercase.
#include <stdio.h>
int main()
{
    char str[50];
    int i;
    printf("Enter string : ");
    gets(str);

    for(i=0; str[i]!='\0'; i++)
    {
        if(str[i]>='a' && str[i]<='z')
        {
            str[i] = str[i] - 32;
        }
    }

    printf("uppercase string is: %s",str);
    return 0;
}
```
### Output
```
Enter string : hello
uppercase string is: HELLO
```
