## C Implementation
### 12. Write a C program to find total number of alphabets, digits or special character in a string.
```c
//Write a C program to find total number of alphabets, digits or special character in a string.
#include <string.h>
#include <stdio.h>
int main()
{
    char str[100];
    int i,alphabets=0,digits=0,specialcharacters=0;
    printf("Enter  the string : ");
    gets(str);
    for(i=0;str[i];i++)
    {
        if((str[i]>=65 && str[i]<=90)|| (str[i]>=97 && str[i]<=122) )
          alphabets++;
        else if(str[i]>=48 && str[i]<=57)
         digits++;
        else
         specialcharacters++;

     }
    printf("Alphabets = %d\n",alphabets);
    printf("Digits = %d\n",digits);
    printf("Special characters = %d", specialcharacters);


    return 0;
}
```
