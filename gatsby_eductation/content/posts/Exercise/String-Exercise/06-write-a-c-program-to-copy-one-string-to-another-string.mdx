## C Implementation
### 6. Write a C program to copy one string to another string.
```c
//Write a C program to copy one string to another string.
#include <stdio.h>
#include <string.h>
int main()
{
  char str1[100],str2[100];

  printf("Enter string ");
  gets(str1);
  strcpy(str2, str1);
  printf("str1 string: %s\n", str1);
  printf("str2 string: %s\n", str2);

  return 0;
}
```
### Output
```
Enter string hello
str1 string: hello
str2 string: hello
```
