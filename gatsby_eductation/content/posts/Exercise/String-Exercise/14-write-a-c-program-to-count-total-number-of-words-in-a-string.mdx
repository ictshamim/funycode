## C Implementation
### 14. Write a C program to count total number of words in a string.
```c
//Write a C program to count total number of words in a string.
#include<stdio.h>
#include <string.h>
int main()
{
    char str[100];
    int i,w=0;

    printf("Enter  the string : ");
    gets(str);

    for(i=0; str[i]; i++)
    {
        if(str[i]==32)
            w++;
    }
    if(i>0)
        w++;

    printf("string has  %d words",w);
    return 0;
}
```
### Output
```
Enter  the string : Hello from funycode
string has  3 words
```
