## C Implementation
### 7. Write a C program to concatenate two strings.
```c
//Write a C program to concatenate two strings.
#include <stdio.h>
#include <string.h>
int main()
{
    char str1[50], str2[50];
    printf("Enter first string: ");
    gets(str1);
    printf("Enter second string: ");
    gets(str2);
    strcat(str1, str2);
      printf("sum of string is string = %s", str1);

    return 0;
}
```

```
Enter first string: hello
Enter second string:  funycode
sum of string is string = hello funycode
```
