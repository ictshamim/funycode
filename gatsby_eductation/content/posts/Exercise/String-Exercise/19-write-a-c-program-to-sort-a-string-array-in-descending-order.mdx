## C Implementation
### 19. Write a C program to sort a string array in descending order.
```c
//Write a C program to sort a string array in descending order.
#include <stdio.h>
#include <string.h>
int main()
{
  char a[100];
  char temp;
  int i,j;
  gets(a);
  for(i=0;i<strlen(a);i++)
  {
      for(j=i+1;j<strlen(a);j++)
  {
      if(a[i]<a[j]){
        temp=a[i];
        a[i]=a[j];
        a[j]=temp;
  }
  }}
  printf("descending order string is:%s",a);
  }
```
### Output
```
Enter a string:acky
ascending order string ykca
```

## Java Implementation

//Write a Java program to sort a string array in descending order.
import java.util.Scanner;
public class Funycode {
    public static void main(String[] args) {
        Scanner z=new Scanner(System.in);
        System.out.println("Enter a string: ");
        String str, dec="";
        int i;
        str = z.nextLine();
        char[] charArray = str.toCharArray();
        String word = new String(charArray);
        for (i=word.length()-1; i >= 0; i--){
            dec += word.charAt(i);
        }

    }
}
