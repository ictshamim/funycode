## C Implementation
### 18. Write a C program to sort a string array in ascending order.
```c
//Write a C program to sort a string array in ascending order.
#include <stdio.h>
#include <string.h>
int main()
{
  char a[100];
  char temp;
  int i,j;
  gets(a);
  for(i=0;i<strlen(a);i++)
  {
      for(j=i+1;j<strlen(a);j++)
  {
      if(a[i]>a[j]){
        temp=a[i];
        a[i]=a[j];
        a[j]=temp;
  }
  }}
  printf("ascending order string is:%s",a);
  }
```
### Output
```
Enter a string:hello
ascending order string ehllo
```
