## C Implementation
### 17. Write a C program to find maximum occurring character in a string.
```c
//Write a C program to find maximum occurring character in a string.
#include <stdio.h>
#include <string.h>
int main()
{
    char a[100];
    int i,j,count=0,maxcount=0;
    char maxcharacter;
    gets(a);
    for(i=0;i<strlen(a);i++)
    {
        count=0;
        for(j=0;j<strlen(a);j++)
    {
        if(a[i]==a[j])
          count++;
    }
    if(maxcount<count)
    {
        maxcount=count;
        maxcharacter=a[i];
    }
    }

    printf("the maximum character is:%c",maxcharacter);

}
```
