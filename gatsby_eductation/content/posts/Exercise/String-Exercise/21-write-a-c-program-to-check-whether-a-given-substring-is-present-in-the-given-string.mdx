## C Implementation
### 21. Write a C program to check whether a given substring is present in the given string.
```c
//Write a C program to check whether a given substring is present in the given string.
#include<stdio.h>
#include<string.h>
int main()
{
     char string[1000], sub[1000];
     int position, length, c = 0;

     printf("Input a string\n");
     gets(string);

     printf("Enter the position and length of substring\n");
     scanf("%d%d", &position, &length);

     while (c < length) {
        sub[c] = string[position+c-1];
        c++;
     }
     sub[c] = '\0';

     printf("Required substring is \"%s\"\n", sub);
}
```

```
Enter a string: hello
Enter the position and length of substring 1 4
Extracted string is: : ello
```
