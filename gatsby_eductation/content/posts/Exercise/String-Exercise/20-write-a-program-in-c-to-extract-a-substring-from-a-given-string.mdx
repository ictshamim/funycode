## C Implementation
### 20. Write a C program to extract a substring from a given string.
```c
//Write a C program to extract a substring from a given string.
#include <stdio.h>
#include <string.h>
#include<stdlib.h>
int main()
{
    char a[50],b[50];
    int index,num,i,l;
    printf("enter the string ");
    gets(a);
    printf("enter start of index and number of characters to extract ");
    scanf("%d%d",&index,&num);
    l=strlen(a);
    if(index+num>l)
    {
        printf("invalid");
        exit(0);
    }
    for(i=0; i<num; i++)
        b[i]=a[index+i];


    b[i]='\0';
    printf("extracted string is:%s",b);

}
```
### Output
```
enter the string hello
enter start of index and number of characters to extract 1 2
extracted string is:el
```

## Java Implementation
### 20. Write a Java program to extract a substring from a given string.
```js
//Write a Java program to extract a substring from a given string.
import java.util.Scanner;

public class Funycode {
    public static void main(String[] args) {
        Scanner z = new Scanner(System.in);
        System.out.println("Enter a string: ");
        String str, result = "";
        int index, num;
        str = z.nextLine();
        System.out.println("enter start of index and number of characters to extract ");
        index = z.nextInt();
        num = z.nextInt();
        if (index + num > str.length()) {
            System.out.println("Invalid");
            System.exit(0);
        }
        for (int i = 0; i < num; i++) {
            result += str.charAt(i);
        }
        System.out.println("Extracted string is: " + result);
    }
}
```
