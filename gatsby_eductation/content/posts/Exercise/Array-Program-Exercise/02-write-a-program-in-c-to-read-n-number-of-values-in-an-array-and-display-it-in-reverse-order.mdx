## C Implementation
### 2. Write a C program to read n number of values in an array and display it in reverse order.
```c
//Write a C program to read n number of values in an array and display it in reverse order.

#include <stdio.h>
void main(){
    int n,r,sum=0,t;
    printf("Input a number: ");
    scanf("%d",&n);
    t=n;
    for( ;n>0; )
    {
         r=n%10;
         sum=sum*10+r;
         n=n/10;
    }
     printf("The number in reverse order is : %d \n",sum);
}
```
### Output
```
Input a number:
123
The number in reverse order is : 321
```
