## C Implementation
### 6. Write a C program to find second largest element in an array.
```c
//Write a C program to find second largest element in an array.
#include <stdio.h>
void main()
{
    int arr[5],i,j=0,lrg,lrg2nd;
    printf("Input 5 elements in the array :\n");
    for(i=0; i<5; i++)
    {
        scanf("%d",&arr[i]);
    }
    lrg=0;
    for(i=0; i<5; i++)
    {
        if(lrg<arr[i])
        {
            lrg=arr[i];
            j = i;
        }
    }
    lrg2nd=0;
    for(i=0; i<5; i++)
    {
        if(i==j)
        {
            i++;
            i--;
        }
        else
        {
            if(lrg2nd<arr[i])
            {
                lrg2nd=arr[i];
            }
        }
    }
    printf("The Second largest element in the array is :  %d ", lrg2nd);
}
```
### Output
```
Input 5 elements in the array :
5
4
3
6
7
The Second largest element in the array is : 6
```
