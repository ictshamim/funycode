## C Implementation
### 3. Write a C program find the sum of all elements of the array.
```c
//Write a C program find the sum of all elements of the array.
#include<stdio.h>
void main()
{
    int a[5],i,j,sum=0;
    printf("input 5 number-\n");
    for(i=0; i<5; i++)
    {
        scanf("%d",&a[i]);
    }
    for(i=0; i<5; i++)
    {
        sum=sum+a[i];
    }
    printf("sum=%d",sum);
}
```
### Output
```
input 5 number-
30
40
10
30
50
sum of all number=160
```
