## C Implementation
### 8. Write a C program to store elements in an array and print it.
```c
//Write a C program to store elements in an array and print it.
#include <stdio.h>
void main()
{
    int arr[5],i,mx,mn;
    printf("Input 5 elements in the array :\n");
       for(i=0;i<5;i++)
        {
          scanf("%d",&arr[i]);
        }
    mx = arr[0];
    mn = arr[0];
    for(i=1; i<5; i++)
    {
        if(arr[i]>mx)
        {
            mx = arr[i];
        }
        if(arr[i]<mn)
        {
            mn = arr[i];
        }
    }
    printf("Maximum element is : %d\n", mx);
    printf("Minimum element is : %d\n\n", mn);
}
```
### Output
```

6
```
