## C Implementation
### 16. Write a C program to delete all duplicate elements from an array.
```c
//Write a C program to delete all duplicate elements from an array.

#include <stdio.h>
int main()
{
  int arr[10], brr[10], count = 0, i, j;
  printf("Enter 10 elements of an array-\n");
  for ( i= 0; i <10; i++)
    scanf("%d", &arr[i]);
  for (i = 0; i <10; i++)
  {
    for (j = 0; j < count; j++)
    {
      if(arr[i] == brr[j])
        break;
    }
    if (j == count)
    {
      brr[count] = arr[i];
      count++;
    }
  }
  printf("Array obtained after removing duplicate elements:\n");
  for (i = 0; i < count; i++)
    printf("%d\n", brr[i]);
}
```
