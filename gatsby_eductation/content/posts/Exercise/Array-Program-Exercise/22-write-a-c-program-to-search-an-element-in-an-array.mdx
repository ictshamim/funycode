## C Implementation
### 21. Write a C program to search an element in an array.
```c
//Write a C program to search an element in an array.
#include<stdio.h>
int main()
{
    int arr[10],i,key;
    printf("Enter 10 elements in array : \n");
    for(i=0; i<10; i++)
    {
        scanf("%d",&arr[i]);
    }
     printf("Enter the key : ");
    scanf("%d", &key);
    for(i=0; i<10; i++)
    {
        if(arr[i]==key)
        {
            printf("element found ");
            return 0;
        }
    }
    printf("element  not  found");
}
```
### Output
```
Enter 10 elements in array :
4 3 2 5 6 7 8 9 4 2
Enter the key : 4
element found
```
