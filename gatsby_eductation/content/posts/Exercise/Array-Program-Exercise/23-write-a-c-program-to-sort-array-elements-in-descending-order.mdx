## C Implementation
### 23. Write a C program to sort array elements in descending order.
```c
//Write a C program to store elements in an array and print it.
#include <stdio.h>
int main()
{
 int i, j, a, arr[10];
 printf("Enter 10 elements- \n");
 for (i = 0; i < 10; i++)
 {
   scanf("%d", &arr[i]);
 }
 for (i = 0; i < 10; ++i)
 {
   for (j = i + 1; j < 10; ++j)
   {
      if (arr[i] < arr[j])
      {
         a =  arr[i];
         arr[i] = arr[j];
         arr[j] = a;
      }
    }
 }
 printf("The numbers arranged in descending order are given below: \n");
 for (i = 0; i < 10; ++i)
        printf("%d\n", arr[i]);
}
```
