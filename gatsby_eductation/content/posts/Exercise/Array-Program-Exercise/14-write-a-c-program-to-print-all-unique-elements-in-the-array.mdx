## C Implementation
### 14. Write a C program to print all unique elements in the array.
```c
//Write a C program to print all unique elements in the array.
#include <stdio.h>
int main()
{
    int arr[10],count=0,i,j,k=10;
    printf("Input 10 elements in the array :\n");
       for(i=0;i<10;i++)
        {
            scanf("%d",&arr[i]);
        }
    printf("\nThe unique elements found in the array are: \n");
    for(i=0; i<10; i++)
    {
        count=0;
        for(j=0; j<k+1; j++)
        {
            if (i!=j)
            {
               if(arr[i]==arr[j])
              {
                 count++;
               }
             }
        }
       if(count==0)
        {
          printf("%d ",arr[i]);
        }
    }
}
```
