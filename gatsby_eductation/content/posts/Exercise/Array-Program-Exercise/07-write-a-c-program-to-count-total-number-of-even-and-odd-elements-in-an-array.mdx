## C Implementation
### 7. Write a C program to count total number of even and odd elements in an array.
```c
//Write a C program to count total number of even and odd elements in an array.
#include<stdio.h>
int main()
{
    int a[10],i,even=0,odd=0;
    printf("Enter 10 Array Elements-\n");
    for(i = 0; i<10; i++)
    {
        scanf("%d", &a[i]);
    }

    for(i = 0; i < 10; i ++)
    {
        if(a[i] % 2 == 0)
        {
            even++;
        }
        else
        {
            odd++;
        }
    }

    printf("Total Number of Even Numbers in this Array = %d \n", even);
    printf("Total Number of Odd Numbers in this Array = %d ", odd);
}
```
