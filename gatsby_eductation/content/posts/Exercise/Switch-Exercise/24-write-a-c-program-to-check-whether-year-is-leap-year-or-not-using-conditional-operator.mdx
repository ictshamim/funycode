## C Implementation
### 24. Write a C program to check whether year is leap year or not using conditional operator.
```c
//Write a C program to check whether year is leap year or not using conditional operator.
#include <stdio.h>
int main()
{
    int y;
    printf("Enter a year: ");
    scanf("%d",&y);
    (y%4==0 && y%100!=0)? printf("leap year"):(y%400==0)?printf("leap year"):printf("not leap year");
}
```
### Output
```
input Enter a year: 2000
output leap year
```
