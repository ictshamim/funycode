## C Implementation
### 22. Write a C program to check whether a number is even or odd using conditional operator.
```c
//Write a C program to check whether a number is even or odd using conditional operator.
#include <stdio.h>
int main()
{
    int n;
    printf("Enter a number: ");
    scanf("%d", &n);
    (n%2 == 0)? printf("number is even"): printf("number is odd");
}
```
### Output
```
input Enter a number: 2
output number is even
```
