## C Implementation
### Program-2: Write a C program to find the maximum between three numbers using switch-case.
```c
/*Write a C program to find the maximum between three numbers using switch-case.*/
#include<stdio.h>
int main()
{
    int a,b,c;
    printf("Input Three Integer:");
    scanf("%d %d %d",&a,&b,&c);
    switch(a>b&&a>c)
    {
        case 1:
             printf("maximum =%d",a);
             break;
        case 0:
             switch(b>c&&b>a)
             {
                case 1:
                      printf("maximum=%d",b);
                      break;
                case 0:
                      printf("maximum=%d",c);
                      break;
              }
        break;
   }
}
```
### Output
```
Input: 3 4 1
Output: maximum number :4
```
