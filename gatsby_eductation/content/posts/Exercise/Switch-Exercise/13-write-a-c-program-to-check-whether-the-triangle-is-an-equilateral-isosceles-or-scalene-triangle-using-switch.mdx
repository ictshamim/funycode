## C Implementation
### 13. Write a C program to check whether the triangle is an equilateral, isosceles or scalene triangle using switch.
```c
//Write a C program to check whether the triangle is an equilateral, isosceles or scalene triangle using switch.
#include<stdio.h>
int main()
{
    int s1,s2,s3;
    printf("Enter the side values:");
    scanf("%d %d %d",&s1,&s2,&s3);
    switch(s1==s2 && s2==s3)
    {
    case 1:
        printf("Equilateral triangle");
        break;
    case 0:
        switch(s1==s2||s1==s3 || s2==s3)
        {
        case 1:
            printf("isosceles triangle");
            break;
        case 0:
            printf("scalene triangle");
            break;
        }
        break;
    }
}
```
### Output
```
Enter the side values:90 30 60
scalene triangle
```
