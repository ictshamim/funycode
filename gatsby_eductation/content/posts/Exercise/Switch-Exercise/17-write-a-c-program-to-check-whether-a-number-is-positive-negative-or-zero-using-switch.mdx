## C Implementation
### 17. Write a C program to check whether a number is positive, negative or zero using switch
```c
//Write a C program to check whether a number is positive, negative or zero using switch
#include <stdio.h>
int main()
{
    int num;
    printf("Enter any number: ");
    scanf("%d", &num);
    switch (num > 0)
    {
    case 1:
        printf("%d is positive", num);
        break;
    case 0:
        switch (num < 0)
        {
        case 1:
            printf("%d is negative", num);
            break;
        case 0:
            printf("%d is zero", num);
            break;
        }
        break;
    }
    return 0;
}
```
### Output
```
input: Enter any number: 6
output: 6 is negative
```
