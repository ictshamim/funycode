## C Implementation
### 14. Write a C program to check whether a number is even or odd using switch case.
```c
//Write a C program to check whether a number is even or odd using switch case.
#include<stdio.h>
int main()
{
    int n;
    printf("Enter a number :");
    scanf("%d",&n);
    switch(n%2)
    {
    case 0:
        printf("Even");
        break;
    case 1:
        printf("Odd");
        break;
    default:
        printf("others");
    }
}
```
### Output
```
Enter a number :3
Odd
```
