## C Implementation
### Program-9: Write a C program to take the value from the user as input any character and check whether it is the alphabet, digit or special character using the switch.
```c
/*Write a C program to take the value from the user as input any character and check whether it is the alphabet, digit or special character using the switch.*/

#include<stdio.h>
int main()
{
    char num;
    printf("Enter a character:");
    scanf("%c",&num);
    switch(num>='a'&& num<='z'||num>='A'&& num<='Z')
    {
        case 1:
            printf("alphabet");
            break;
        case 0:
            switch(num>='0' && num<='9')
            {
                case 1:
                    printf("It's digit");
                    break;
                case 0:
                    printf("special character");
                    break;
            }
            break;
    }
}
```
### Output
```
Input: a
Output: alphabet
```
