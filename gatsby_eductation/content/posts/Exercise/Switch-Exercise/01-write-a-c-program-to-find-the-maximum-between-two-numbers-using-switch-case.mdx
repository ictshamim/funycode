## C Implementation
### Program-1: Write a C program to find the maximum between two numbers using switch-case.
```c
/*Write a C program to find the maximum between two numbers using switch-case.*/
#include<stdio.h>
void main()
{
   int a,b;
   printf("Enter Number:");
   scanf("%d %d",&a,&b);
   switch(a > b)
   {
      case 0:
         printf("maximum number %d",b);
         break;
      case 1:
         printf("maximum number %d",a);
         break;
      default:
         printf("Both are equal");
   }
}
```
### Output
```
Input: 3 4
Output: maximum number :4
```
