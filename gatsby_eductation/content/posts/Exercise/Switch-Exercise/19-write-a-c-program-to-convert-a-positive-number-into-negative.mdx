## C Implementation
### 19. Write a C program to convert a positive number into negative number and negative number into a positive number using switch.
```c
//Write a C program to convert a positive number into negative number and negative number into a positive number using switch.
#include<stdio.h>
int main()
{
    float a;
    int value;
    scanf("%d",&value);
    switch(value)
    {
    case 1:
        a=a*(-1);
        printf("the Positive form of the number is");
        scanf("%d",&a);
        break;
    case 2:
        a=a*(-1);
        printf("Negative form of the number is");
        scanf("%d",&a);
        break;
    default:
        printf("Invalid Input");
    }
}
```
### Output
```
Hello, World!
```
