## C Implementation
### Program-6: Write a C program to check whether a year is a leap year or not using switch.
```c
/*Write a C program to check whether a year is a leap year or not using switch*/
#include<stdio.h>
int main()
{
    int y,n;
    printf("Enter Year: ");
    scanf("%d",&y);
    n=((y%4==0)&&((y%400==0)||(y%100!=0)));
    switch(n)
    {
       case 1:
           printf("Leap Year.");
           break;
       case 0:
           printf("Not Leap Year.");
           break;
       default:
           printf("Invalid.");
           break;
    }
    return 0;
}
```
### Output
```
Input: 2000
Output: Leap Year.
```
