## C Implementation
### 10. Write a C program to check whether a character is an uppercase or lowercase alphabet. using the switch.
```c
//Write a C program to check whether a character is an uppercase or lowercase alphabet. using the switch.
#include<stdio.h>
int main()
{
    char ch;
    printf("Enter a character:");
    scanf("%c",&ch);
    switch(ch>='a'&& ch<='z')
    {
    case 1:
        printf("lowercase");
        break;
    case 0:
        switch(ch>='A' && ch<='Z')
        {
        case 1:
            printf("Uppercase");
            break;
        case 0:
            printf("Not Alphabet");
            break;
        }
        break;
    }
}
```
### Output
```
input: Enter a character:d
output: lowercase
```
