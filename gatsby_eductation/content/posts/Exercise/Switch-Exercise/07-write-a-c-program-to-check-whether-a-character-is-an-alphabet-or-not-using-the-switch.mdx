## C Implementation
### Program-7: Write a C program to check whether a character is an alphabet or not using the switch.
```c
/*Write a C program to check whether a character is an alphabet or not using the switch.*/
#include<stdio.h>
int main()
{
    char ch;
    printf("Enter a character: ");
    scanf("%d",&ch);
    switch((ch>='a'&&ch<='z') || (ch>='A'&&ch<='Z'))
    {
    case 0:
        printf("Alphabet");
        break;
    case 1:
        printf("not Alphabet");
        break;
    default:
        printf("Invalid input");
    }
}
```
### Output
```
Input: a
Output: Alphabet
```
