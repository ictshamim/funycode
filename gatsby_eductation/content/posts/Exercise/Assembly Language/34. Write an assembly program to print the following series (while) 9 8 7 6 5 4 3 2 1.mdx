### 34. Write an assembly program to print the following series (while) 9 8 7 6 5 4 3 2 1

```
.model small
.stack 100h
.data
mg1 db ?
mg2 db ?
.code
main proc

 mov ax,@data
 mov ds,ax

 mov ah,1
 int 21h
 sub al,30h
 mov bl,al

 while_:
 cmp bl,0
 je exit
 mov ah,2
 mov cl,bl
 add cl,30h
 mov dl,cl
 int 21h
 dec bl
 jmp while_

 exit:
 mov ah,4ch
 int 21h

 main endp
 end main
```

## Input
```
9
```
## Output
```
9 8 7 6 5 4 3 2 1
```
