### 24. Write an assembly program to find small number between two numbers.
```
.model small
.stack 100h
.data
mg1 db ?
mg2 db ?
.code
main proc

 mov ax,@data
 mov ds,ax

 mov ah,1
 int 21h
 sub al,30h
 mov mg1,al

 mov ah,1
 int 21h
 sub al,30h
 mov mg2,al

 cmp al,mg1
 jg less
 jl greater

 less:
 mov ah,2
 add mg1,30h
 mov dl,mg1
 int 21h
 jmp exit

 greater:
 mov ah,2
 add mg2,30h
 mov dl,mg2
 int 21h
 jmp exit

 exit:
 mov ah,4ch
 int 21h

 main endp
 end main
```

## Input
```
2
5
```
## Output
```
2
```
