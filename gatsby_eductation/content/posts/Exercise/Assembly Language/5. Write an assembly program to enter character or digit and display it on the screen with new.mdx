### 5. Write an assembly program to enter character or digit and display it on the screen with new line
```
.model small
.stack 100h
.data

.code
main proc

 mov ah,1
 int 21h

 mov bl,al

 mov ah,2
 mov dl,13
 int 21h

 mov ah,2
 mov dl,10
 int 21h

 mov ah,2
 mov dl,bl
 int 21h

 mov ah,4ch
 int 21h

 main endp
 end main
```

## Input
```
1
```
## Output
```
1
```
