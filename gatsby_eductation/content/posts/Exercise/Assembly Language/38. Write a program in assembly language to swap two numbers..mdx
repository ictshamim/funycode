### 38. Write a program in assembly language to swap two numbers.
```
.MODEL SMALL
.STACK 100H
.CODE
MAIN PROC
        MOV AX,'6'
        MOV BX,'8'
        PUSH AX
        PUSH BX
        POP  AX
        POP BX
        MOV DX,AX
        MOV AH,2
        INT 21H
        MOV DX,BX
        INT 21H
        MOV AH,4CH
        INT 21H
        MAIN ENDP
        END MAIN
```


## Output
```
86
```
