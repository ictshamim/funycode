### 1. Write an assembly language program to print a character.
```
.model small
.stack 100h
.data
.code
main proc
 mov ah,2
 mov dl,"A"
 int 21h

 mov ah,4ch
 int 21H

 main endp
 end main
```

## Output
```
A
```
