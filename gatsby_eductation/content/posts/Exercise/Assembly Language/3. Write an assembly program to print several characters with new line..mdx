### 3. Write an assembly program to print several characters with new line
```
.model small
.stack 100h
.data

.code
main proc
 mov ah,2
 mov dl,"A"
 int 21h

 mov ah,2
 mov dl,13
 int 21h

 mov ah,2
 mov dl,10
 int 21h

 mov ah,2
 mov dl,'B'
 int 21h

 mov ah,4ch
 int 21h

 main endp
 end main
```


## Output
```
A
B
```
