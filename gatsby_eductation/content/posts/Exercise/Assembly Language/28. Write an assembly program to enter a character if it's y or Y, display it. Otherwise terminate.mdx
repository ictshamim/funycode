### 28. Write an assembly program to enter a character, if it's y or Y, display it, Otherwise terminate
```
.model small
.stack 100h
.data
mg1 db ?
mg2 db ?
.code
main proc

 mov ax,@data
 mov ds,ax

 mov ah,1
 int 21h
 mov mg1,al

 cmp al,'y'
 je display_
 cmp al,'Y'
 je display_
 jmp exit

 display_:
 mov ah,2
 mov dl,al
 int 21h
 jmp exit

 exit:
 mov ah,4ch
 int 21h

 main endp
 end main
```

## Input
```
y
```
## Output
```
y
```
