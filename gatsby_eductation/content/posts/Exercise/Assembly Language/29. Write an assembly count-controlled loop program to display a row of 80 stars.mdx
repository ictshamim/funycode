### 29. Write an assembly count-controlled loop program to display a row of 80 stars.
```
.model small
.stack 100h
.data
mg1 db ?
mg2 db ?
.code
main proc

 mov ax,@data
 mov ds,ax

 mov cx,80
 jcxz skip
 top:
  mov ah,2
  mov dl,'*'
  int 21h
  loop top
  jmp skip

  skip:
 mov ah,4ch
 int 21h

 main endp
 end main
```
