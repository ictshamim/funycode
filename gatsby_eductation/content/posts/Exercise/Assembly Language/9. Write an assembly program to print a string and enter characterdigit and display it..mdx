### 9. Write an assembly program to print a string and enter character/digit and display it
```
.model small
.stack 100h
.data
msg1 db "Enter a number:$"
msg2 db 13,10,"You Entered:$"
.code
main proc

 mov ax,@data
 mov ds,ax

 lea dx,msg1
 mov ah,9
 int 21h

 mov ah,1
 int 21h

 ;lea dx,msg2
 mov dx, offset msg2
 mov ah,9
 int 21h

 mov ah,2
 mov dl,al
 int 21h



 mov ah,4ch
 int 21h
 main endp
 end main
```

## Input
```
Enter a number 7
```
## Output
```
You Entered: 7
```
