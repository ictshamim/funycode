## C Implementation
### 6. Write a C program to declare, initialize an union, example of union.
```c
//Write a C++ program to declare, initialize an union, example of union.
#include<stdio.h>
union student{
    char name[100];
};
int main(){
    union student p;
    strcpy(p.name, "Ahosanul Hasan ");
    printf("%s\n",p.name);
}
```
### Output
```
Ahosanul Hasan
```
