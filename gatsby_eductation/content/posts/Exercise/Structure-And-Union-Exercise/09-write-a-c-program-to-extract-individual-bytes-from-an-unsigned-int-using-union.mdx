## C Implementation
### 9. Write a C program to extract individual bytes from an unsigned int using union.
```c
//Write a C++ program to extract individual bytes from an unsigned int using union.
#include<stdio.h>
#include <string.h>
union tagname
{
    unsigned int a;
    unsigned char s[4];
};
int main()
{
    union tagname object;
    char i;
    object.a=0xAABBCCDD;
    printf("Integer number: %ld, hex: %X\n",object.a,object.a);
    printf("Indivisual bytes: ");
    for(i=3;i>=0;i--)
        printf("%02X ",object.s[i]);
    printf("\n");
}
```

```

Indivisual bytes: � �  �
```
