## C Implementation
### 1. Write a C program to print first 50 natural numbers using recursion.
```c
//Write a C program to print first 50 natural numbers using recursion.
#include<stdio.h>
struct student{
    char name[100];
    int id;
};
int main(){
    struct student sakib;
    strcpy(sakib.name, "Ahosanul Hasan");
    sakib.id=16105001;
    printf("%s\n",sakib.name);
    printf("%d",sakib.id);
}
```
### Output
```
Ahosanul Hasan
16105001
```

## C++ Implementation
### 1. Write a C++ program to print first 50 natural numbers using recursion.
```c
//Write a C++ program to print first 50 natural numbers using recursion.

#include<iostream>
#include <string.h>
using namespace std;
struct student{
    char name[100];
    int id;
};
int main(){
    struct student sakib;
    strcpy(sakib.name, "Ahosanul Hasan");
    sakib.id=16105001;
    cout << sakib.name << endl;
    cout << sakib.id;
}
```
