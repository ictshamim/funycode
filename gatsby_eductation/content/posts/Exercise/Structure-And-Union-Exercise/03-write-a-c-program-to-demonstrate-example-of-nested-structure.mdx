## C Implementation
### 3. Write a C program to demonstrate example of nested structure.
```c
//Write a C++ program to demonstrate example of nested structure.
#include<stdio.h>
#include<string.h>
struct student{
    char name[100];
    int id;
    int class_;
    struct subject{
    float mark;
    };
    struct subject bangla,english,math;
};
int main(){
   struct student one;
   strcpy(one.name, "Ahosanul Hasan");
   one.id=12;
   one.class_=7;
   one.bangla.mark=90;
   printf("Name   : %s\n",one.name);
   printf("class  : %d\n",one.class_);
   printf("id     : %d\n",one.id);
   printf("Bangla : %f\n",one.bangla.mark);
}
```
### Output
```
Name   : Ahosanul Hasan
class  : 7
id     : 12
Bangla : 90
```
