## C Implementation
### 3. Write a C program to test whether a character entered is a small case letter or not using macro definition.
```c
//Write a C program to test whether a character entered is a small case letter or not using macro definition.
#include <stdio.h>
#define MAX(x,y) ((x>y)?x:y)
int main()
{
    int a,b,max;
    printf("Enter first number: ");
    scanf("%d",&a);
    printf("Enter second number: ");
    scanf("%d",&b);
    max=MAX(a,b);
    printf("Maximum number is: %d\n",max);
    return 0;
}
```
### Output
```
Enter first number: 2 4
Enter second number: Maximum number is: 4
```
