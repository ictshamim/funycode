## C Implementation
### 6. Write a C program to convert a uppercase alphabet to lowercase.
```c
//Write a C program to convert a uppercase alphabet to lowercase.
#include <stdio.h>
#define UtoL(a)  a+32;
int main()
{
    char ch;
    printf("Enter the Uppercase Alphabet: ");
    scanf("%c",&ch);
    ch = UtoL(ch);
    printf("Lowercase of the Alphabet is: %c \n",ch);
    return 0;
}
```
### Output
```
Enter the Uppercase Alphabet: A
Lowercase of the Alphabet is: a
```
