## C Implementation
### 1. Write a C program to test whether a character entered is a small case letter or not using macro definition.
```c
//Write a C program to test whether a character entered is a small case letter or not using macro definition.
#include <stdio.h>
// Macro definitions
#define IS_LOWER(x) (x >= 'a' && x <= 'z')
int main()
{
    char ch;
    // Input a character from user
    printf("Enter any character: ");
    ch = getchar();
    if (IS_LOWER(ch))
        printf("'%c' is lowercase\n", ch);
    else
        printf("Entered character is not lowercase\n");
    return 0;
}
```
### Output
```
Enter any character: a
'a' is lowercase
```
