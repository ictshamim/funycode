## C Implementation
### 17. Write a C program to find power of any number using functions
```c
//Write a C program to find power of any number using functions
#include<stdio.h>

int power(int a,int b)
{
    int total=1;
    while(b!=0)
    {
        total*=a;
        b--;
    }
    return total;
}

int main()
{
    int base,exp;
    printf("Enter base: ");
    scanf("%d",&base);
    printf("Enter exponent: ");
    scanf("%d",&exp);

    int ans=power(base,exp);

    printf("Ans is: %d\n",ans);
    return 0;
}
```
### Output
```
Enter base: 1
Enter exponent: 5
Ans is: 1
```
