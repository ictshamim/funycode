## C Implementation
### 33. Write a C program to Find the Volume and Surface Area of Sphere
```c
//Write a C program to Find the Volume and Surface Area of Sphere
#include<stdio.h>
#include<math.h>

float surfaceAreaSphere(float r)
{
    float pi=3.14159;
    return 4*pi*r*r;
}
float volumeSphere(float r)
{
    float pi=3.14159;
    return (4/3.0)*pi*r*r*r;
}

int main()
{
    float radius;
    printf("Enter the radius: ");
    scanf("%f",&radius);

    float surface=surfaceAreaSphere(radius);
    float volume=volumeSphere(radius);

    printf("Surface Area of this cuboid is %.2f\n",surface);
    printf("Surface Area of this cuboid is %.2f\n",volume);
    return 0;
}
```
### Output
```
Enter the radius: 5
Surface Area of this cuboid is 314.16
Surface Area of this cuboid is 523.60
```
