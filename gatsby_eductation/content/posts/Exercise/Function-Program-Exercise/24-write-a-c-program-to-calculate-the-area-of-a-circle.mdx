## C Implementation
### 24. Write a C program to Calculate the Area of a Circle
```c
//Write a C program to Calculate the Area of a Circle
#include<stdio.h>

float circleArea(float r)
{
    float pi=3.14159;

    return pi*r*r;
}

int main()
{
    float radius;
    printf("Enter radius: ");
    scanf("%f",&radius);

    float area=circleArea(radius);

    printf("Area is %.2f\n",area);
    return 0;
}
```
### Output
```
Enter radius: 4
Area is 50.27
```

//Write a Java program to Calculate the Area of a Circle
import java.util.Scanner;
