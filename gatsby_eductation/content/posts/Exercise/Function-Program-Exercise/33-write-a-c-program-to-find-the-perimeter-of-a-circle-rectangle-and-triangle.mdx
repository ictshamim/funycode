## C Implementation
### 34. Write a C program to Find the Perimeter of a Circle, Rectangle and Triangle
```c
//Write a C program to Find the Perimeter of a Circle, Rectangle and Triangle
#include<stdio.h>

void perimeterOfCircle()
{
    float pi=3.14159,r;
    printf("Enter radius: ");
    scanf("%f",&r);
    float perimeter=2*pi*r;
    printf("Permeter of circle is %.2f: \n",perimeter);
}

void perimeterOfRectangle()
{
    int a,b;
    printf("Enter base & height: ");
    scanf("%d%d",&a,&b);
    float perimeter=2*(a+b);
    printf("Permeter of Rectangle is %.2f: \n",perimeter);
}

void perimeterOfTriangle()
{
    float a,b,c;
    printf("Enter 3 sides of triangle: ");
    scanf("%f%f%f",&a,&b,&c);
    float perimeter=a+b+c;
    printf("Permeter of circle is %.2f: \n",perimeter);
}

int main()
{
    perimeterOfCircle();
    perimeterOfRectangle();
    perimeterOfTriangle();
}
```
### Output
```
Enter radius: 3
Permeter of circle is 18.85:
Enter base & height: 4 5
Permeter of Rectangle is 18.00:
Enter 3 sides of triangle: 5 5 5
Permeter of circle is 15.00:
```
