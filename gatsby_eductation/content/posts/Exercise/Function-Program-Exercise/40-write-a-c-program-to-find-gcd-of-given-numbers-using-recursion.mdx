## C Implementation
### 40. Write a C program to find GCD of given Numbers using Recursion
```c
//Write a C program to find GCD of given Numbers using Recursion
#include<stdio.h>

int findHCF(int a,int b)
{
    if(b==0)
        return a;
    return findHCF(b,a%b);
}

int main()
{
    int a,b;
    printf("Enter two number: ");
    scanf("%d%d",&a,&b);
    int hcf=findHCF(a,b);
    printf("HCF is %d\n",hcf);
    return 0;
}
```
### Output
```
Enter two number: 1 2
HCF is 1
```
