## C Implementation
### 21. Write a C program to Calculate the Area of a Triangle
```c
//Write a C program to Calculate the Area of a Triangle
#include<stdio.h>
#include<math.h>

float triangleArea(float a,float b,float c)
{
    float s,area;
    s=(a+b+c)/2;

    area=sqrt(s*(s-a)*(s-b)*(s-c));
    return area;
}

int main()
{
    float a,b,c,s;
    printf("Enter 3 sides: ");
    scanf("%f%f%f",&a,&b,&c);

    float area=triangleArea(a,b,c);

    printf("Area is %.2f\n",area);
    return 0;
}
```
### Output
```
Enter 3 sides: 90 45 45
Area is 0
```
