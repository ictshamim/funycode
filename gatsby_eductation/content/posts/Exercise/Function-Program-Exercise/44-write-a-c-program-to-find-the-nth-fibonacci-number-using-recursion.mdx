## C Implementation
### 44. Write a C program to find the Nth Fibonacci Number using Recursion
```c
//Write a C program to find the Nth Fibonacci Number using Recursion#include<stdio.h>
#include<stdio.h>

int fib(int n)
{
    if(n<=1)
        return n;
    return fib(n-1)+fib(n-2);
}

int main()
{
    int n;
    printf("Enter nth : ");
    scanf("%d",&n);

    int value=fib(n);

    printf("%dth fibonacchi number is: %d\n",n,value);
    return 0;
}
```
### Output
```
Enter nth : 10
10th fibonacchi number is: 55
```
