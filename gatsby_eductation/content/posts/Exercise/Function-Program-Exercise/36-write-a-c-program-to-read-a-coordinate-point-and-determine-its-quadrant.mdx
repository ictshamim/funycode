## C Implementation
### 36. Write a C program to Read a Coordinate Point and Determine its Quadrant
```c
//Write a C program to Read a Coordinate Point and Determine its Quadrant
#include<stdio.h>

int checkQuardent(int x,int y)
{
    if(x==0 && y==0)
        printf("This point is in center\n");
    else if(x==0)
        printf("This point is in Y axis\n");
    else if(y==0)
        printf("This point is in X axis\n");
    else if(x>0 && y>0)
        printf("This point is in first Quardent\n");
    else if(x<0 && y>0)
        printf("This point is in Second Quardent\n");
    else if(x<0 && y<0)
        printf("This point is in Third Quardent\n");
    else
        printf("This point is in Fourth Quardent\n");

}

int main()
{
    int x,y;
    printf("Enter X & Y: ");
    scanf("%d%d",&x,&y);
    checkQuardent(x,y);
    return 0;
}
```
### Output
```
Enter X & Y: 5 10
This point is in first Quardent
```
