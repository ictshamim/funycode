## C Implementation
### 28. Write a C program to Compute the Surface Area & Volume of a Cube
```c
//Write a C program to Compute the Surface Area & Volume of a Cube
#include<stdio.h>

int surfaceAreaCube(int edge)
{
    return 6*edge*edge;
}

int volumeCube(int edge)
{
    return edge*edge*edge;
}

int main()
{
    int edge;
    printf("Enter the length of an edge: ");
    scanf("%d",&edge);

    int surface=surfaceAreaCube(edge);
    int volume=volumeCube(edge);

    printf("Surface Area of this cube is %d\n",surface);
    printf("Volume of this cube is %d\n",volume);
    return 0;
}
```
### Output
```
Enter the length of an edge: 10
Surface Area of this cube is 600
Volume of this cube is 1000
```
