## C Implementation
### 2. Write a C program to print all even or odd numbers in given range using functions .
```c
//Write a C program to print all even or odd numbers in given range using functions .
#include<stdio.h>

void evenOrodd(int n)
{
    printf("Odd numbers are: ");
    for(int i=1;i<=n;i+=2)
    {
        printf("%d ",i);
    }
    printf("\nEven numbers are: ");
    for(int i=2;i<=n;i+=2)
    {
        printf("%d ",i);
    }
}

int main()
{
    int n;
    printf("Enter n: ");
    scanf("%d",&n);

    evenOrodd(n);

    return 0;
}
```
### Output
```
Enter n: 10
Odd numbers are: 1 3 5 7 9
Even numbers are: 2 4 6 8 10
```
