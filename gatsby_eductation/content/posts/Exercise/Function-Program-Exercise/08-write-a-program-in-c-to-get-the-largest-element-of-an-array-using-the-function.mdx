## C Implementation
### 8. Write a C program to get the largest element of an array using the function.
```c
//Write a C program in C to get the largest element of an array using the function.
#include<stdio.h>

int findlarge(int ara[],int n)
{
    int mx=-1;
    for(int i=0;i<n;i++)
    {
        if(mx<ara[i])
            mx=ara[i];
    }
    return mx;
}
int main()
{
    int n;
    printf("Enter array size: ");
    scanf("%d",&n);
    int ara[n];
    for(int i=0;i<n;i++)
    {
        scanf("%d",&ara[i]);
    }

    int large=findlarge(ara,n);

    printf("%d is largest element\n",large);
    return 0;
}
```
### Output
```
Enter array size: 5
1 2 3 4 5
5 is largest element
```
