## C Implementation
### 39. Write a C program to find LCM of a Number using Recursion
```c
//Write a C program to find LCM of a Number using Recursion
#include<stdio.h>

int findLCM(int a,int b)
{
    static int tmp=1;
    if(tmp%a==0 && tmp%b==0)
    {
        return tmp;
    }
    else
    {
        tmp++;
        findLCM(a,b);
        return tmp;
    }
}

int main()
{
    int a,b;

    printf("Enter two number: ");
    scanf("%d%d",&a,&b);
    int lcm=findLCM(a,b);
    printf("HCF is %d\n",lcm);

    return 0;
}
```
### Output
```
Enter two number: 1 2
HCF is 1
```
