## C Implementation
### 47. Write a C program to find sum of all even or odd numbers in given range using functions
```c
//Write a C program to find sum of all even or odd numbers in given range using functions
#include<stdio.h>

int evenSum(int n)
{
    int sum=0;
    for(int i=2;i<=n;i+=2)
    {
        sum+=i;
    }
    return sum;
}

int main()
{
    int n;
    printf("Enter n: ");
    scanf("%d",&n);

    int total=evenSum(n);

    printf("Total even sum is %d",total);
    return 0;
}
```
### Output
```
Enter n: 10
Total even sum is 30
```
