## C Implementation
### 48. Write a C program to find reverse of any number using functions .
```c
//Write a C program to find reverse of any number using functions .
#include<stdio.h>

int reverse(int n)
{
    int tmp=0;
    while(n!=0)
    {
        int x=n%10;
        tmp=tmp*10+x;
        n/=10;
    }
    return tmp;
}

int main()
{
    int n;
    printf("Enter n: ");
    scanf("%d",&n);

    int rev=reverse(n);

    printf("Reverse number is %d",rev);
    return 0;
}
```
### Output
```
Enter n: 10
Reverse number is 1
```
