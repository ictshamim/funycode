## C Implementation
### 29. Write a C program to Find the Volume and Surface Area of cylinder
```c
//Write a C program to Find the Volume and Surface Area of cylinder
#include<stdio.h>

float surfaceAreaCylinder(float radius,float height)
{
    float pi=3.14159;
    return 2*pi*radius*height+2*pi*radius*radius;
}

int main()
{
    float radius,height;
    printf("Enter the radius & height: ");
    scanf("%f%f",&radius,&height);

    float surface=surfaceAreaCylinder(radius,height);

    printf("Surface Area of this cube is %.2f\n",surface);
    return 0;
}
```
### Output
```
Enter the radius & height: 5 10
Surface Area of this cube is 471.24
```
