## C Implementation
### 18. Write a C program to check whether a number is palindrome or not using functions .
```c
//Write a C program to check whether a number is palindrome or not using functions .
#include<stdio.h>

void checkPalindrome(int n)
{
    int num=0,a=n;
    int tmp;
    while(a!=0)
    {
        tmp=a%10;
        num=num*10+tmp;
        a/=10;
    }
    if(n==num)
        printf("%d is palindrome.\n",n);
    else
        printf("%d is not palindrome\n",n);
}

int main()
{
    int n;
    printf("Enter a number: ");
    scanf("%d",&n);

    checkPalindrome(n);

    return 0;
}
```
### Output
```
Enter a number: 4
4 is palindrome.
```
