## C Implementation
### 3. Write a C program in C to find the square of any number using the function.
```c
//Write a C program in C to find the square of any number using the function.
#include <stdio.h>
void SwapValue(int a, int b) {
   int t = a;
   a = b;
   b = t;

   printf("\nAfter swapping, the values are: a = %d, b = %d", a, b);
}
int main() {
   int a, b;
   printf("Enter value of a & b : ");
   scanf("%d%d", &a,&b);
   SwapValue(a,b);
   return 0;
}
```
### Output
```
Enter value of a & b : 2 4

After swapping, the values are: a = 4, b = 2
```
