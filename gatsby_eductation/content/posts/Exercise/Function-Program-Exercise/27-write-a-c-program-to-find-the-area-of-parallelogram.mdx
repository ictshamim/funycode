## C Implementation
### 27. Write a C program to find the Area of Parallelogram
```c
//Write a C program to find the Area of Parallelogram
#include<stdio.h>

float parallelogram(float b,float h)
{
    return b*h;
}

int main()
{
    float base,height,area;
    printf("Enter base & height of a parallelogram: ");
    scanf("%f%f",&base,&height);

    area=parallelogram(base,height);

    printf("Area of this parallelogram is %.2f\n",area);
    return 0;
}
```
### Output
```
Enter base & height of a parallelogram: 5 5
Area of this parallelogram is 25.00
```
