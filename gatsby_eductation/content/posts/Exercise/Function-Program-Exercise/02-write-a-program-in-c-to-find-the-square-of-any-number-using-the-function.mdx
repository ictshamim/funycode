## C Implementation
### 2. Write a C program to find the square of any number using the function.
```c
//Write a C program to find the square of any number using the function.
#include<stdio.h>

int square(int num)
{
    return num*num;
}

int main()
{
    int n;
    printf("Enter a number: ");
    scanf("%d",&n);

    int ans=square(n);

    printf("%d\n",ans);
    return 0;
}
```
### Output
```
Enter a number: 4
16
```

### 2. Write a Java program to find the square of any number using the function.

import java.util.Scanner;
