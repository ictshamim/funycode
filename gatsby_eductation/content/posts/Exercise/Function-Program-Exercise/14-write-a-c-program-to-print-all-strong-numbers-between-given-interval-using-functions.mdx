## C Implementation
### 14. Write a C program to print all strong numbers between given interval using functions.
```c
//Write a C program to print all strong numbers between given interval using functions.
#include<stdio.h>

int fact(int n)
{
    int total=1;
    for(int i=1;i<=n;i++)
        total*=i;
    return total;
}

void strongNumber(int a,int b)
{
    int tmp,num,r;
    for(int i=a;i<=b;i++)
    {
        int sum=0;
        num=i;
        while(num)
        {
            r=num%10;
            sum=sum+fact(r);
            num=num/10;
        }
        if(sum==i)
            printf("\nThe number %d is a strong number",i);
        else
            printf("\nThe number %d is not a strong number",i);
    }
}

int main()
{
    int a,b;
    printf("Enter a range: ");
    scanf("%d%d",&a,&b);

    strongNumber(a,b);

    return 0;
}
```
### Output
```
Enter a range: 1 5

The number 1 is a strong number
The number 2 is a strong number
The number 3 is not a strong number
The number 4 is not a strong number
The number 5 is not a strong number
```
