## C Implementation
### 4. Write a C program to check a given number is even or odd using the function.
```c
//Write a C program to check a given number is even or odd using the function.
#include<stdio.h>

int isEvenorOdd(int num)
{
    if(num%2==0)
        printf("%d is even.\n",num);
    else
        printf("%d is odd.\n",num);
}

int main()
{
    int n;
    printf("Enter a number: ");
    scanf("%d",&n);

    isEvenorOdd(n);

    return 0;
}
```
### Output
```
Enter a number: 5
5 is odd.
```

## Java Implementation
### 4. Write a Java program to check a given number is even or odd using the function.

//Write a Java program to show the simple structure of a function.
import java.util.Scanner;
public class Funycode {
    static void isEvenorOdd(int num)
    {
        if(num%2==0)
            System.out.println(num + " is even.\n");
        else
            System.out.println(num + " is odd.\n");
    }

    public static void main(String[] args) {
        Scanner z = new Scanner(System.in);
        int n;
        System.out.println("Enter a number: ");

        isEvenorOdd(n);
    }
}
