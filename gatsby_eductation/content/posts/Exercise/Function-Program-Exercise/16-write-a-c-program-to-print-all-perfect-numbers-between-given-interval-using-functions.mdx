## C Implementation
### 16. Write a C program to print all perfect numbers between given interval using functions.
```c
//Write a C program to print all perfect numbers between given interval using functions.
#include<stdio.h>

void Perfect(int a,int b)
{
    for(int i=a;i<=b;i++)
    {
        int Number=i;
        int Sum=0;
        for(int i = 1 ; i < Number ; i++)
        {
            if(Number % i == 0)
                Sum = Sum + i ;
        }
        if (Sum == Number)
            printf("%d is a Perfect Number\n", Number) ;
        else
            printf("%d is not the Perfect Number\n", Number) ;
    }
}


int main()
{
    int a,b;
    scanf("%d%d",&a,&b);

    Perfect(a,b);

    return 0;
}
```
### Output
```
Enter a number: 4
16
```
