## C Implementation
### 6. Write a C program to convert decimal number to binary number using the function.
```c
//Write a C program in C to convert decimal number to binary number using the function.
#include<stdio.h>

int convDecimaltoBinary(int n)
{
    int bin=0;
    int i=1;
    while(n!=0)
    {
        int rem=n%2;
        bin += rem*i;
        i*=10;
        n/=2;
    }
    return bin;
}

int main()
{
    int n;
    scanf("%d",&n);

    int bin=convDecimaltoBinary(n);

    printf("Sum is: %d\n",bin);

    return 0;
}
```
### Output
```
Enter a number: 4
16
```
