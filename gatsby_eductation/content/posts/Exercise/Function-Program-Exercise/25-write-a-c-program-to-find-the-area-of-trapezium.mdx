## C Implementation
### 25. Write a C program to Find the Area of Trapezium
```c
//Write a C program to Find the Area of Trapezium
#include<stdio.h>

int square(int num)
{
    return num*num;
}

int main()
{
    int n;
    printf("Enter a number: ");
    scanf("%d",&n);

    int ans=square(n);

    printf("%d\n",ans);
    return 0;
}
```

## C++ Implementation
### 25. Write a C++ program to find the square of any number using the function.
```c
//Write a C++ program to find the square of any number using the function.

#include<iostream>
using namespace std;
float trapeziumArea(int p1,int p2,int h)
{
    return 0.5*(p1+p2)*h;
}
int main()
{
    float parallel1,parallel2,height,area;
    cout << "Enter Parallel sides: ";
    cin >> parallel1 >>parallel2;
    cout << "Enter height: ";
    cin >> height;
    area=trapeziumArea(parallel1,parallel2,height);
    cout << "Area is <<area;
}
```

```

Enter height: 7
Area is 52.50
```
