## C Implementation
### 35. Write a C program to Find the Sum of First N Natural Numbers
```c
//Write a C program to Find the Sum of First N Natural Numbers
#include<stdio.h>

int sum(int n)
{
    return n*(n+1)/2;
}

int main()
{
    int n;
    printf("Enter n: ");
    scanf("%d",&n);
    int total=sum(n);
    printf("Sum is %d\n",total);
    return 0;
}
```
### Output
```
Enter n: 5
Sum is 15
```
