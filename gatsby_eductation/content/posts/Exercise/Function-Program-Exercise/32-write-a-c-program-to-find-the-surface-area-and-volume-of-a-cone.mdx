## C Implementation
### 32. Write a C program to Find the Surface Area and Volume of a Cone
```c
//Write a C program to Find the Surface Area and Volume of a Cone
#include<stdio.h>
#include<math.h>

float surfaceAreaCone(float r,float h)
{
    float pi=3.14159;
    return pi*r*(r+sqrt(r*r+h*h));
}
float volumeCone(float r,float h)
{
    float pi=3.14159;
    return pi*r*r*(h/3.0);
}

int main()
{
    float radius,height;
    printf("Enter the radius & height: ");
    scanf("%f%f",&radius,&height);

    float surface=surfaceAreaCone(radius,height);
    float volume=volumeCone(radius,height);

    printf("Surface Area of this cuboid is %.2f\n",surface);
    printf("Surface Area of this cuboid is %.2f\n",volume);
    return 0;
}
```
### Output
```
Enter the radius & height: 10 15
Surface Area of this cuboid is 880.52
Surface Area of this cuboid is 1570.79
```
