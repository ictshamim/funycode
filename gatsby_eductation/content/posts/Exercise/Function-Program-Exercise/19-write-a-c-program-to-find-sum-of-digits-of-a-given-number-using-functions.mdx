## C Implementation
### 19. Write a C program to find sum of digits of a given number using functions .
```c
//Write a C program to find sum of digits of a given number using functions .
#include<stdio.h>

int sumofDigits(int n)
{
    int sum=0;
    while(n!=0)
    {
        sum=sum+n%10;
        n=n/10;
    }
    return sum;
}

int main()
{
    int n;
    printf("Enter a number: ");
    scanf("%d",&n);

    int sum=sumofDigits(n);

    printf("Total: %d\n",sum);

    return 0;
}
```
### Output
```
Enter a number: 12
Total: 3
```
