## C Implementation
### 26. Write a C program to Find the Area of Rhombus
```c
//Write a C program to Find the Area of Rhombus
#include<stdio.h>

int main()
{
    float d1,d2,area;
    printf("Enter 2 diagonal of a Rhombus: ");
    scanf("%f%f",&d1,&d2);

    area=0.5*d1*d2;

    printf("Area of this Rhombus is %.2f\n",area);
    return 0;
}
```
### Output
```
Enter 2 diagonal of a Rhombus: 5 5
Area of this Rhombus is 12.50
```
