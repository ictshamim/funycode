## C Implementation
### 1. Write a C program to show the simple structure of a function.
```c
//Write a C program to show the simple structure of a function.
#include<stdio.h>

void introduction()
{
    printf("Hi\n");
    printf("I am Funycode");
}

int main()
{
    introduction();
    return 0;
}
```
### Output
```
Hi
I am Funycode
```
