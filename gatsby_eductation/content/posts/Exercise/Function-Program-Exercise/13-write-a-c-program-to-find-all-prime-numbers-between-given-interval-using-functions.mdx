## C Implementation
### 13. Write a C program to find all prime numbers between given interval using functions.
```c
//Write a C program to find all prime numbers between given interval using functions.
#include<stdio.h>

void findPrime(int a,int b)
{
    int n;
    printf("Prime numbers are: ");
    for(int k=a;k<=b;k++)
    {
        int check=1;
        n=k;
        for(int i=2;i*i<=n;i++)
        {
            if(n%i==0)
            {
                check=0;
                break;
            }
        }
        if(check && n!=1)
            printf("%d ",n);
    }
}

int main()
{
    int a,b;
    printf("Enter a range: ");
    scanf("%d%d",&a,&b);

    findPrime(a,b);

    return 0;
}
```
### Output
```
Enter a range: 1 5
Prime numbers are: 2 3 5
```
