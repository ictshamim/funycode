## C Implementation
### 31. Write a C program to Find the Volume and Surface Area of Cuboids
```c
//Write a C program to Find the Volume and Surface Area of Cuboids
#include<stdio.h>

int surfaceAreaCuboid(int length,int breadth,int height)
{
    return 2*(length*breadth+breadth*height+height*length);
}

int main()
{
    int length,breadth,height;
    printf("Enter the length, breadth & height: ");
    scanf("%d%d%d",&length,&breadth,&height);

    int surface=surfaceAreaCuboid(length,breadth,height);

    printf("Surface Area of this cuboid is %d\n",surface);
    return 0;
}
```
### Output
```
Enter the length, breadth & height: 10 5 7
Surface Area of this cuboid is 310
```
