## C Implementation
### 9. Write a C program to find the largest element using Dynamic Memory Allocation.
```c
//Write a C program to find the largest element using Dynamic Memory Allocation.
#include <stdio.h>
void main()
{
    int i,n;
    float *p;
    printf(" Enter the number of elements= ");
    scanf("%d",&n);
    p=(float*)calloc(n,sizeof(float));  // assigning memory

    if(p==NULL)
    {
        printf(" No memory is allocated.");
        exit(0);
    }
    printf("\n Enter the elements\n");
    for(i=0;i<n;++i)
    {
       scanf("%f",p+i);
    }
    for(i=1;i<n;++i)
    {
       if(*p<*(p+i))
           *p=*(p+i);
    }
    printf(" The Largest element is= %.2f ",*p);
}
```
### Output
```
 Enter the number of elements= 3

 Enter the elements
 1 2 3
 The Largest element is= 3.00
```
