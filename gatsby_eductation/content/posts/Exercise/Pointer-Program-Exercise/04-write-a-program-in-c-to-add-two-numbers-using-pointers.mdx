## C Implementation
### 4. Write a C program to add two numbers using pointers.
```c
//Write a C program to add two numbers using pointers.
#include <stdio.h>
void main()
{
    int a, b, *p1, *p2, sum;

    printf("Enter two integers\n");
    scanf("%d%d", &a, &b);

    p1 = &a;
    p2 = &b;

    sum = *p1 + *p2;

    printf("Sum of the numbers = %d\n", sum);
}
```
### Output
```
Enter two integers
1 2
Sum of the numbers = 3
```
## C++ Implementation
### 4. Write a C++ program to add two numbers using pointers.
```c
//Write a C++ program to add two numbers using pointers.
#include <iostream>
using namespace std;
int main()
{
    int a, b, *p1, *p2, sum;

    cout<<("Enter two integers\n");
    cin>>a>>b;

    p1 = &a;
    p2 = &b;

    sum = *p1 + *p2;

    cout<<"Sum of the numbers = "<< sum;
}
```
### Output
```
Enter two integers
1 2
Sum of the numbers = 3
```
