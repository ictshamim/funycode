## C Implementation
### 19. Write a C program to show a pointer to union.
```c
//Write a C program to show a pointer to union.
#include <stdio.h>
void main()
{
    union student
    {
        char *name;
    };
    union student stu,*p;
    stu.name="abcde";  //assign  the string
    p=&stu;
    printf(" %s \n",(*p).name);
}
```
### Output
```
abcde
```
