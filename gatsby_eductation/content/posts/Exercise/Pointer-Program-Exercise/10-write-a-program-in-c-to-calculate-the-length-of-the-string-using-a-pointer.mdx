## C Implementation
### 10. Write a C program to Calculate the length of the string using a pointer.
```c
//Write a C program to Calculate the length of the string using a pointer.
#include<stdio.h>
void main() {
  char s[1000], *p;
  int count = 0;
  printf("Enter Any string = ");
  scanf("%s",s);
  p= s;
  while (*p != '\0') {
    count++;
    p++;
  }
  printf("Length of String : %d", count);
}
```
### Output
```
Enter Any string = hello
Length of String : 5
```
