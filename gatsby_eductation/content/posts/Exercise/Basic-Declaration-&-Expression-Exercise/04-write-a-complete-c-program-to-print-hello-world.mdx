## C Implementation
### Program-4: Write a C program to print your name, date of birth and mobile number.
```c
/*Write a C program that will print your name, your father and mother name in three separate lines. */
#include <stdio.h>
 int main()
  {
     printf("rifia tamanna\n");
     printf("4 october 2000\n");
     printf("0175555555\n");
     return(0);
  }
```

```
rafia tamnanna
4 october 2000
0175555555
```
