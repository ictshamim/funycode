## C Implementation
### Program-22: Write a C program to convert this temperature into Centigrade degrees.
```c
/*Write a C program to convert this temperature into Centigrade degrees.*/
#include<stdio.h>
void main()
{
    float f,c,d;
    scanf("%f",&f);
    d=f-32;
    c=(d/9)*5;
    printf("Celsius=%f",c);
}
```
### Output
```
20
Celsius:=-6.666667
```
