## C Implementation
### Program-26: Write a C Program to interchange values of two numbers without using third variable
```c
/*Write a C Program to interchange values of two numbers without using third variable*/
#include<stdio.h>
void main()
{
    int a,b;
    scanf("%d%d",&a,&b);
    printf("A=%d B=%d\n",a,b);
    a=a+b;
    b=a-b;
    a=a-b;
    printf("A=%d B=%d",a,b);
}
```
### Output
```
5
10
A:=5 B=10
A:=10 B=5
```
