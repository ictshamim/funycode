## C Implementation
### 1. Write a C program to print Hello World.
```c
//Write a C program to print Hello World.
#include <stdio.h>
int main()
{
// printf() displays the string inside quotation
    printf("Hello, World!");
    return 0;
}
```
### Output
```
Hello, World!
```

## C++ Implementation
### 1. Write a C++ program to print Hello World.
```c
//Write a C++ program to print Hello World.
#include<iostream>
using namespace std;
int main()
{
    cout<<"hello world";
    return 0;
}
```
### Output
```
Hello, World!
```
