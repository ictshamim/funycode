## C Implementation
### 15. Write a C program that calculates the volume of a sphere.
```c
/*Write a C program that calculates the volume of a sphere.*/
#include<stdio.h>
void main()
{
    int a;
    double b,c;
    scanf("%d",&a);
    c=4/3;
    b=c*3.1416*a*a*a;
    printf("volume=%f",b);
}
```
### Output
```
10
volume=3141.6000000000004
```
