## C Implementation
### 10. Write a C program to enter two numbers and perform all arithmetic operations.
```c
//Write a C program to enter two numbers and perform all arithmetic operations.
#include<stdio.h>
void main()
{
    int a,b,c,d,x,y;
    scanf("%d%d",&x,&y);
    a=x+y;
    b=x-y;
    c=x*y;
    d=x/y;
    printf("add=%d\n",a);
    printf("sub=%d\n",b);
    printf("multi=%d\n",c);
    printf("divi=%d\n",d);
}
```
### Output
```
input : 2 2
output:
add=4
sub=0
multi=4
divi=1
```
