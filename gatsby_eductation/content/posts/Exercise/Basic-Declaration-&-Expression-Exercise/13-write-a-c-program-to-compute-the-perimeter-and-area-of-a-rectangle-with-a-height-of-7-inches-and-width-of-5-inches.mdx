## C Implementation
### Program-14: Write a C Program to accept any character from user and display its ASCII number on screen.
```c
/*Write a C Program to accept any character from user and display its ASCII number on screen.*/
#include<stdio.h>
void main()
{
    char a;
    scanf("%c", &a);
    printf("Ascii value %c = %d",a,a);
}
```
### Output
```
A
Ascii value 65
```
