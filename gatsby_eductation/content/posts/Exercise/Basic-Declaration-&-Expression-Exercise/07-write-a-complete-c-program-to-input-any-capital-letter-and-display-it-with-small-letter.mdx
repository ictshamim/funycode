## C Implementation
### Program-8: Write a complete C Program to input any Capital letter and display it with small letter.
```c
/*Write a complete C Program to input any Capital letter and display it with small letter. */
#include <stdio.h>
void main()
{
    char lower, upper;
    printf("Please input a lowercase character: ");
    scanf("%c", &lower);
    if(lower >= 'a' && lower <= 'z'){
         upper = ('A' + lower - 'a');
    }
    else{
        upper = lower;
    }
    printf("\nThe uppercase equivalent is: %c", upper);
}
```
### Output
```
Input: Please input a lowercase character: a
Output: The uppercase equivalent is: A
```
