## C Implementation
### Program-24: Write a C program to calculate the area & perimeter of the rectangle, and the area & circumference of the circle.
```c
/*Write a c program to calculate the area & perimeter of the rectangle, and the area & circumference of the circle.*/
#include<stdio.h>
void main()
{
    int a,b,c,d,e;
    float f,g;
    scanf("%d%d",&a,&b);
    c=2*(a+b);
    d=a*b;
    printf("Rectangle Perimeter=%d\n",c);
    printf("Rectangle Area=%d\n",d);
    scanf("%d",&e);
    f=2*3.1416*e;
    g=3.1416*e*e;
    printf("Circle Circumference=%f\n",f);
    printf("Circle Area=%f",g);
}
```
### Output
```
5
6

Rectangle Area:=30
5
Circle Circumference:=31.416
Circle Area:=78.54
```
