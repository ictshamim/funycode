## C Implementation
### Program-6: Write a complete C program to return the absolute value of an integer.
```c
/*Write a complete C program to return the absolute value of an integer. */
#include<stdio.h>
void main()
{
    int n;
    scanf("%d",&n);
    printf("the absulate value=%d",abs(n));
}
```
### Output
```
Input: -3
Output: 3
```
