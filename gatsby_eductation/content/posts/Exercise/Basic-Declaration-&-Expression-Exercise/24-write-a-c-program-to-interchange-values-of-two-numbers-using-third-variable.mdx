## C Implementation
### Program-25: Write a C Program to interchange values of two numbers using third variable.
```c
/*Write a C Program to interchange values of two numbers using third variable.*/
#include<stdio.h>
void main()
{
    int a,b,c;
    scanf("%d%d",&a,&b);
    printf("A=%d B=%d\n",a,b);
    c=a;
    a=b;
    b=c;
    printf("A=%d B=%d",a,b);
}
```
### Output
```
5
10
a:=5 b=10
a:=10 b=5
```
