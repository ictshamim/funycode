## C Implementation
### Program-12: Write a C program to declare two integers and one float variables then initialize them to 10, 15, and 12.6. It then prints these values on the screen.
```c
/*Write a C program to declare two integers and one float variables then initialize them to 10, 15, and 12.6. It then prints these values on the screen.*/
#include<stdio.h>
void main()
{
    int a,b;
    float c;
    a=10;
    b=15;
    c=12.6;
    printf("A=%d B=%d C=%f",a,b,c);
}
```
### Output
```
A=10
B=15
C=12.6
```
