## C Implementation
### Program-23: Write a C program to find the third angle of a triangle if two angles are given.
```c
/*Write a C program to find the third angle of a triangle if two angles are given.*/
#include<stdio.h>
void main()
{
    int a,b,c;
    scanf("%d%d",&a,&b);
    c=180-(a+b);
    printf("Third Angle=%d",c);
}
```
### Output
```
50
50
Third Angle:=80
```
