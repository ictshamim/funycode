## C Implementation
### Program-11: Write a C program to find the quotient and remainder of a division
```c
/*Write a C program to find the quotient and remainder of a division. */
#include<stdio.h>
void main()
{
  int a,b,c,d;
  scanf("%d%d",&a,&b);
  c=a/b;
  d=a%b;
  printf("%d %d",c,d);
}
```
### Output
```
Input: 10 5
Output: 2 0
```
