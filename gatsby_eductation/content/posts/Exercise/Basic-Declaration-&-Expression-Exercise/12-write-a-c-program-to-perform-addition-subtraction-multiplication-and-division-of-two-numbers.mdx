## C Implementation
### Program-13: Write a C program to perform addition, subtraction, multiplication and division of two numbers.
```c
/*Write a C program to perform addition, subtraction, multiplication and division of two numbers.*/
#include<stdio.h>
void main()
{
    int a,b,c,d,x,y;
    printf("Enter two Number");
    scanf("%d%d",&x,&y);
    a=x+y;
    b=x-y;
    c=x*y;
    d=x/y;
    printf("add=%d\n",a);
    printf("sub=%d\n",b);
    printf("multi=%d\n",c);
    printf("divi=%d\n",d);
}
```
### Output
```
Enter two Number
10
2
add=12
sub=8
Multi=20
Div=5
```
